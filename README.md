# Terraria 服务器监控

AstrBot 插件，用于监控 Terraria TShock 服务器状态，支持玩家上下线通知、在线查询与远程指令执行。

## 功能

- **玩家上下线通知** — 定时轮询服务器，玩家加入/离开时自动推送到群聊
- **在线状态查询** — 使用 `/ss` 查看当前在线玩家
- **远程指令执行** — 管理员使用 `/tc <命令>` 执行服务器命令（需配置白名单）
- **多群支持** — 可同时推送到多个群，多个群可使用指令

## 前置要求

- TShock 服务器已启用 REST API
- 已在 TShock 中创建 REST API 用户

## 配置项

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `tshock_host` | TShock REST API 地址（含端口，如 `http://1.2.3.4:7878`） | `""` |
| `tshock_username` | REST API 用户名 | `""` |
| `tshock_password` | REST API 密码 | `""` |
| `session_ids` | 推送目标群的 session ID 列表（使用 `/sid` 获取） | `[]` |
| `group_ids` | 允许使用指令的群 ID 列表 | `[]` |
| `admin_ids` | 允许执行 `/tc` 命令的管理员 ID 列表 | `[]` |
| `poll_interval` | 轮询间隔（秒） | `15` |

## 指令

| 指令 | 说明 | 权限 |
|------|------|------|
| `/ss` | 查看服务器在线玩家 | 配置群内所有人 |
| `/tc <命令>` | 执行 TShock 服务器命令（如 `/tc who`） | 仅管理员白名单 |
